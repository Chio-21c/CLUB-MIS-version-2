# Gardens Estate Secondary School Club MIS System

A full-stack **Club Management Information System (MIS)** built with **React (Vite)**, **Express/Node.js**, **Sequelize/PostgreSQL**, and **Tailwind CSS**.  
This system allows students to join clubs, admins to manage memberships, and patrons to oversee club activities.

---

##  Features

### Student Dashboard
- Gradient background (blue → yellow).
- Animated carousel of club images with names scrolling continuously.
- "Join a Club" button that opens a membership form modal.
- Membership form with admission number, name, class, gender, and club selection.

### Admin Dashboard
- Secure login for admins.
- Approve or reject membership requests.
- Delete members.
- Edit member details inline.
- Reports: Members per club, all members with clubs.
- Export reports to CSV.

### Patron Dashboard
- ➕ Add, edit, and delete patrons.
- 📋 View patrons with club assignments.
- 🔍 Search and filter patrons.

### Club Management
- ➕ Add, edit, and delete clubs.
- 📋 View registered clubs.
- 🔍 Search and filter clubs.

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL with Sequelize ORM
- **Auth:** JWT-based authentication
- **Styling:** Tailwind CSS (responsive, modern UI)
